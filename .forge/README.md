# .forge — Agent Automation Assets

Prompts, workflows, and templates for AI-assisted development. Not runtime code.

| Path | Purpose |
|------|---------|
| `prompts/spec-creation.md` | Draft a feature spec |
| `prompts/task-breakdown.md` | Break spec into BACKLOG tasks |
| `prompts/implementation.md` | Implement one atomic task |
| `prompts/context-update.md` | Refresh CONTEXT.md |
| `workflows/phase-0.md` | Phase 0 implementation order |
| `templates/handoff.md` | Mid-task agent handoff |

Load via `docs/CONTEXT_MAP.md` → Agent prompts row.
